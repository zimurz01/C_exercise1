# C_exercise1
A quick exercise for my C exam, Computer Engineering.
The main purpose is to create a program that can record student's data dinamically. I respected ADT and used dynamic memory allocation.
On top of that the program allows you to sort the students by their average with a bubble sort algorythm, for now it only sorts from lower to higher average
but in the future i might make it dynamic allowing a keyboard input to decide how to sort the data.
